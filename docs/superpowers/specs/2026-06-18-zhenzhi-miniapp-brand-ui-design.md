# 真知格斗小程序品牌 UI 设计记录

## 目标

按方案二执行：用户端小程序做完整品牌刷新，运营端小程序做轻量品牌同步。当前阶段不重做后台网站，不改变约课、取消、登录、排课、消课和会员绑定流程。

## 参考依据

- 品牌名：真知格斗 / ZHENZHIGEDOU。
- Logo：黑白红拳手形象，红色拳套是主视觉记忆点。
- 场地照片：黑灰训练地面、橙色训练垫和墙面、暖色灯带、白银色拳袋、干净训练和休息区。

## 视觉方向

- 主底色使用黑灰，保持拳馆训练空间的力量感。
- 主行动色使用品牌红 `#e31b23`，用于预约、确认、选中和危险确认。
- 辅助强调色使用橙色 `#f28a24`，对应训练垫和暖光，用于边线、标签、品牌线和运营指标强调。
- 卡片继续保持 8px 圆角，符合现有小程序视觉系统，不改成营销页风格。
- 用户端更有品牌感，运营端更克制，优先保证扫描、录入和重复操作效率。

## 覆盖范围

- 共享品牌组件：使用本地 `zhenzhi-logo.jpg`，显示 `真知格斗 / ZHENZHIGEDOU`。
- 课程首页：训练场开放中的品牌标题、红黑橙训练场背景、课程筛选和课程卡统一品牌色。
- 预约页：预约卡和取消按钮加入橙色训练垫边线和品牌红操作色。
- 课程详情页：详情 hero、信息分区、教练卡和预约按钮统一红橙品牌处理。
- 账户页：登录面板、会员资料、运营入口、规则入口和消课记录同步品牌样式。
- 运营端小程序：运营 hero、门店/分区导航、指标卡、表单和操作按钮同步品牌色，但不牺牲工具密度。

## 验证

- 新增 `pnpm miniapp:brand-ui:test`，覆盖 Logo 资产、共享品牌组件、全局主题、首页品牌文案、详情/账户/预约页品牌色。
- 仍需在微信开发者工具中做多设备视觉走查，特别确认小屏幕下标题、按钮和表单没有溢出。
